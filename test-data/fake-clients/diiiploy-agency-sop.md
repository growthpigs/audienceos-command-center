## 1. Client Onboarding Process

### 1.1 Pre-Engagement

**Sales Handoff Checklist:**
- [ ] Signed SOW received
- [ ] Payment terms confirmed
- [ ] Primary contacts identified
- [ ] Communication preferences documented
- [ ] Access credentials requested
- [ ] Kickoff call scheduled

**Timeline:** Complete within 48 hours of contract signature

### 1.2 Kickoff Call Agenda

1. **Introductions** (5 min)
   - Team members and roles
   - Communication channels

2. **Goals Review** (15 min)
   - Confirm objectives from SOW
   - Define success metrics
   - Establish baseline numbers

3. **Access & Setup** (10 min)
   - Platform access status
   - Required credentials
   - Technical requirements

4. **Timeline Review** (10 min)
   - Phase milestones
   - Check-in cadence
   - Deliverable schedule

5. **Q&A** (10 min)

### 1.3 Client Folder Structure

Every client gets a standardized folder in Drive:

```
/Clients/[ClientName]/
├── 00-Contracts/
│   ├── SOW.pdf
│   └── Amendments/
├── 01-Strategy/
│   ├── Client-Brief.md
│   ├── ICP-Definition.md
│   └── Competitive-Analysis.md
├── 02-Campaigns/
│   ├── Outbound/
│   ├── Paid-Media/

├── 03-Assets/
```

## 2. Project Management

### 2.1 Task Management

**Tool:** Linear
**Workspace:** Diiiploy

**Project Structure:**
- One project per client per major engagement
- Cycles align with calendar months
- Labels for priority and type

**Task States:**
| State | Meaning |
|-------|---------|
| Backlog | Not started, not prioritized |
| Todo | Prioritized for current cycle |
| In Progress | Actively being worked |
| In Review | Awaiting client/team review |
| Done | Completed and verified |

### 2.2 Time Tracking

**All billable time must be logged within 24 hours.**

Categories:
- Strategy & Planning
- Campaign Setup
- Creative Development
- Optimization & Reporting
- Client Communication
- Admin

### 2.3 Communication Standards

**Response Times:**
| Channel | Response Time |
|---------|---------------|
| Slack (urgent) | 2 hours |
| Slack (normal) | Same business day |
| Email | 24 hours |
| Voicemail | 4 hours |

**Meeting Notes:**
- Document key decisions within 24 hours
- Share with all attendees
- Store in client's Communications folder
- Include action items with owners

## 3. Service Delivery

### 3.1 Outbound System Setup

**Phase 1: Foundation (Week 1)**
1. Audit existing outreach (if any)
2. Define/refine ICP
3. Set up tech stack (Clay, Instantly, LinkedIn)
4. Create email domains and warm-up

**Phase 2: List Building (Week 2)**
1. Build initial prospect list (500-1000)
2. Enrich data
3. Score and segment
4. Validate email deliverability

**Phase 3: Campaign Launch (Week 3)**
1. Write and approve sequences
2. Set up A/B tests
3. Launch initial campaigns
4. Monitor deliverability

**Phase 4: Optimization (Week 4+)**
1. Analyze response rates
2. Iterate on messaging
3. Scale winning sequences
4. Weekly optimization cycles

### 3.2 Paid Media Setup

**Account Audit:**
- Historical performance review
- Audience analysis
- Creative assessment
- Conversion tracking verification

**Campaign Structure:**
```
Campaign Level: Objective (Conversions, Traffic, etc.)
└── Ad Set Level: Audience targeting
    └── Ad Level: Creative variations
```

**Naming Convention:**
`[Client]_[Platform]_[Objective]_[Audience]_[Date]`

Example: `UrbanThread_Meta_Conv_LAL1%_2025-01`

### 3.3 Reporting Cadence

| Report Type | Frequency | Delivery Day |
|-------------|-----------|--------------|
| Performance Snapshot | Weekly | Monday |
| Deep Dive Analysis | Monthly | 5th |
| QBR Deck | Quarterly | 15th |

## 4. Quality Assurance

### 4.1 Campaign QA Checklist

Before any campaign goes live:

**Copy & Creative:**
- [ ] Spelling/grammar checked
- [ ] Brand guidelines followed
- [ ] CTAs clear and actionable
- [ ] Links tested and tracking

**Technical:**
- [ ] UTM parameters correct
- [ ] Conversion tracking verified
- [ ] Audience targeting confirmed
- [ ] Budget limits set

**Approval:**
- [ ] Internal review complete
- [ ] Client approval received
- [ ] Launch date confirmed

### 4.2 Error Handling

**If a mistake is made:**
1. Stop the affected campaign immediately
2. Assess impact (spend, data, reputation)
3. Notify client within 1 hour
4. Document root cause
5. Implement fix
6. Create prevention measures
7. Update SOPs if needed

## 5. Client Success

### 5.1 Health Scoring

| Score | Status | Criteria |
|-------|--------|----------|
| Green | Healthy | Goals on track, communication good |
| Yellow | At Risk | Missing targets or communication gaps |
| Red | Critical | Significant issues, escalation needed |

Review health scores weekly in Monday standup.

### 5.2 Escalation Path

1. **Level 1:** Account Manager handles directly
2. **Level 2:** Chase involvement for strategy issues
3. **Level 3:** Executive discussion for contract/relationship issues

### 5.3 Renewal Process

**90 days before renewal:**
- Review engagement results
- Prepare renewal proposal
- Schedule strategy call

**60 days before renewal:**
- Present renewal options
- Negotiate terms if needed
- Obtain verbal commitment

**30 days before renewal:**
- Send contract
- Process payment
- Plan next phase

## 7. Compliance & Ethics

### 7.1 Data Handling

- Never share client data between clients
- PII must be stored securely
- Honor opt-out requests within 24 hours
- Maintain CAN-SPAM compliance

### 7.2 Financial Services Clients

Additional requirements:
- All copy requires compliance review
- No performance guarantees
- Testimonials need written consent
- Maintain records for 7 years

### 7.3 Confidentiality

- Client strategies are confidential
- Don't discuss client names without permission
- Case studies require written approval
- NDA terms must be honored
